JPL Solar System Services (astroquery.solarsystem.jpl)¶

This submodule is a collection of Solar System-related services provided by the Jet Propulsion Laboratory (JPL). Modules, classes, and functions in this submodule can be imported as part of this submodule or directly from their top-level modules.

  • JPL SBDB Queries (astroquery.jplsbdb/astroquery.solarsystem.jpl.sbdb)
  • JPL Horizons Queries (astroquery.jplhorizons/astroquery.solarsystem.jpl.horizons)
