list_cats(typ, *, fout=None, ignore_noncrit=False)[source]¶

List catalogs in given database.

Listing contains:

  2. Cone search access URL

  3. Warning codes

  4. Warning descriptions

Parameters:
typstr

Any value in self.dbtypes.

foutoutput stream

Default is screen output.

ignore_noncritbool

Exclude warnings in astroquery.vo_conesearch.validator.conf.noncritical_warnings. This is useful to see why a catalog failed validation.
